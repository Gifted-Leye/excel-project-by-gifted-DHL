# excel-project-by-gifted-DHL
Excel data modelling and data cleaning completed

This Excel projecct shows the dataset from raw and uncleaned to cleaned version of the sales dataset. I used TRIM, PROPER, and IF functions to remove inconsistencies, handle missing values, and standardize text formats. The cleaned data improved accuracy and made further analysis reliable. 
i also used pivot tabless and built interactive dashboard that summarizes monthly sales performance. It highlights top-performing products and revenue trends, helping stakeholders make informed decisions.
